49er Developers Full Stack Application

Overview

I created this application from scratch over the course of a semester in the Network Based Application Development course I took this spring.

The main purpose of the website is to give UNC Charlotte students a place to create and view upcoming events related to computer science courses and other development related topics.

The website allows users to sign up and login to an account; create, view, update, and delete connection events. The user and event data is stored persistently utilizing a database.

Technologies

The application utilizes Node.js, Express, and MongoDB, and a few other packages for the back-end functionality. The application is modularized using the Model-View-Controller (MVC) architecture.

The front-end of the website was created utilizing HTML5 (EJS), CSS, and JavaScript. The application utilizes media queries to make the layout responsive across a variety of screen sizes including desktop, mobile, and tablet devices.

Security

When creating an account user passwords are hashed and salted, so in the event the database is compromised, the plaintext user passwords would not be exposed. Additionaly, when signing up, the email address entered must be unique to prevent an attacker from creating another account with the same email address as a user.

Users must be logged in to an account to create, edit, or delete a post. Only authenticated users that have a matching session ID are able to edit and delete the event posts they created.

All fields that take user input are validated and sanitized before being stored to prevent cross-site scripting (XSS) attacks.
